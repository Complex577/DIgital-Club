# Digital ID System - Quick Start Guide

## For Members

### Getting Your Digital ID

Your digital ID is generated automatically when you:
1. Log in for the first time after system update
2. Update your profile information

No manual action needed!

### Accessing Your ID

**From Dashboard:**
```
Login → Dashboard → "My Digital ID" card → View Full ID
```

**From Profile:**
```
Login → Profile → Digital ID Card section → View Full ID
```

**Direct URL:**
```
http://your-site.com/member/digital-id
```

### Downloading Your ID

1. Go to your Digital ID page
2. Click "Download ID Card" button
3. Save the PNG image
4. Transfer to your phone for easy access

### Using Your ID at Events

1. Open the downloaded ID image on your phone
2. Show it to event organizers
3. They can scan the QR code to verify your membership
4. Or they can manually check your ID number

## For Event Organizers

### Verifying a Member ID

**Method 1: Scan QR Code**
1. Use any QR code scanner app
2. Point camera at the QR code on the ID
3. Automatically opens verification page
4. Check for green "VALID MEMBER" badge

**Method 2: Manual Verification**
1. Ask member for their ID number (e.g., DC-2025-0001)
2. Visit: `http://your-site.com/verify-id/DC-2025-0001`
3. Check verification result

**What You'll See:**
- ✅ Valid: Green badge, member photo, name, course, status
- ❌ Invalid: Red badge, error message

## Troubleshooting

### ID Not Showing
- **Solution**: Log out and log back in
- System will auto-generate on next login

### Need to Update ID
- Update your profile (name, photo, course, etc.)
- ID regenerates automatically
- Or click "Regenerate ID" button

### Can't Download ID
- Make sure you're logged in
- Check browser download settings
- Try "View Full ID" first, then download

### QR Code Not Working
- Ensure you have internet connection
- Try manual verification with ID number
- Contact admin if issue persists
